🗂️ GL Rater - Test Data Manager

This utility helps you populate test quotes in your browser's localStorage for the GL Rater application.

Instructions:

  1. Open this file in the same browser where you use the GL Rater app
  2. Click "Add Test Quotes" to populate sample data
  3. Navigate to http://localhost:3000/rating/history to see the quotes
  4. Use "Clear All Quotes" to reset if needed

Note: The data is stored in your browser's localStorage and will persist until cleared.
